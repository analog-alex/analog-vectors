# Wiki Source

This directory keeps the GitHub Wiki content in the main repository so documentation changes can be reviewed in pull requests.

## Purpose

Provide a reviewable source of truth for longer-form documentation that should live in the GitHub Wiki instead of `README.md`.

## Audience

Maintainers and contributors updating tutorials, onboarding guides, and workflow documentation.
